# 프로젝트 설치 및 실행

1. 저장소 클론

```shell
git clone https://github.com/cham-min/wanted-pre-onboarding-frontend.git
```

2. 프로젝트 패키지 설치

```shell
npm install
```

3. 프로젝트 실행

```shell
npm start
```
